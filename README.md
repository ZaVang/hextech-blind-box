# Hextech Blind Box 🔮

LOL选手盲盒抽卡系统

## 数据结构

选手数据支持分版本描述，同一选手不同时期可有不同评级和标签。

```json
{
  "name": "Tian",
  "position": "打野",
  "versions": [
    {"era": "2019", "team": "FPX", "rating": "SSS", "tags": ["冠军血脉", "节奏大师", "操作型", "FMVP"]},
    {"era": "2021", "team": "FPX", "rating": "S", "tags": ["冰岛往事", "神经刀", "节奏大师"]},
    {"era": "2023-2024", "team": "TES", "rating": "SS", "tags": ["节奏大师", "神经刀"]}
  ]
}
```

## 评级体系

- **SSS**: 历史级表现，冠军/FMVP
- **SS**: 顶级选手，稳定C
- **S**: 一流选手，关键先生
- **A**: 优秀选手，稳健
- **B**: 及格线

## 羁绊标签

包含约25种羁绊标签，如：
- 冠军血脉、冰岛往事、操作型、节奏大师、神经刀、外战幻神等

## 抽卡规则（规划中）

- 随机抽取5个位置各1人
- 按羁绊标签聚合
- 评级加权计算阵容强度

## 数据来源

基于最近8年MSI/S赛八强队伍选手信息整理
